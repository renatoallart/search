## Installation

Clone the project

```bash
  git clone https://github.com/renatoallart/search.git
```

Go to the project directory

```bash
  cd search
```

Install dependencies

```bash
  npm install or yarn add
```

Start the server

```bash
  npm run dev or yarn dev
```
